## **ðŸ“œIntroduction**
* In this project we coded from scratch an AirBnb clone.
* This repository contains a collection of Python scripts that come together to create a functional console interface for a file storage system. The system supports both interactive and non-interactive modes, and can be used either directly through the console, or imported into other scripts to augment their functionality.
